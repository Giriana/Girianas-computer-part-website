What is a Computer Network?

A computer network is a system that connects multiple computers and devices to share resources and communicate with each other. Networks can range from small home setups to vast global systems like the internet.

Networking Components

Key components of a computer network include:

  • Nodes: Devices connected to the network (computers, printers, servers)
  • Network Interface Cards (NIC): Hardware that connects devices to the network
  • Switches & Hubs: Devices that connect multiple devices on a network
  • Routers: Devices that connect different networks together
  • Cables/Wireless: The medium for data transmission
  • Protocols: Rules for communication (TCP/IP, HTTP, FTP)

Network Addressing

Network addressing helps identify devices and route data:

  • MAC Address: Unique hardware identifier for network interfaces
  • IP Address: Logical address assigned to each device (IPv4 like 192.168.1.1 or IPv6)
  • DNS: Translates domain names (google.com) to IP addresses
  • Port Numbers: Identify specific services or applications

Network Types

Networks can be categorized by size and purpose:

  • PAN (Personal Area Network): Very small network around an individual
  • LAN (Local Area Network): Small geographic area like home or office
  • WAN (Wide Area Network): Spans large distances (the internet is the largest WAN)
  • MAN (Metropolitan Area Network): City-wide network
  • VPN (Virtual Private Network): Secure network over public infrastructure
