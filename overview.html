Networking Overview

Networking Components

A computer network is made up of several key components:

  • Nodes (computers, printers, servers)
  • Network Interface Cards (NICs)
  • Switches and Routers
  • Cables or wireless connections
  • Protocols (rules for communication)

Network Addressing

Every device on a network needs an address to communicate:

  • IP Address (like 192.168.1.1)
  • MAC Address (physical hardware address)

Network Benefits

Computer networks provide many advantages:

  • Resource sharing (printers, files)
  • Communication (email, messaging)
  • Internet access
  • Centralized data storage

Network Types

There are different types of networks:

  • LAN (Local Area Network)
  • WAN (Wide Area Network)
  • MAN (Metropolitan Area Network)
  • PAN (Personal Area Network)
